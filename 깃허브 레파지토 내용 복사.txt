INFORMATION_RENEW sourcetree에 clone 시키기고 vscode 실행시키기

새로운 파일 하나 만든다 -> sourcetree에 clone에서 INFORMATION_RENEW 레파지토리 주소 복사하고 새로 만든 파일을 경로로 잡고 클론한다. -> 새폴더 안에 들어가서 cmd 해서 vscode 실행시킨다. 



INFORMATION_RENEW 에서 vscode로 작업 내용 옮기는 방법 


INFORMATION_RENEW에서 zip 파일을 다운로드 받는다 -> 빈 폴더를 하나 만든다 -> sourcetree에 clone에서  vscode 레파지토리 주소 붙여놓고 빈 파일로 경로를 잡는다 -> 그러면 빈 폴더에 vscode 레파지토리에 올라가있던 것이 생성이 되어있을 것이다 -> zip 파일을 압축해제한다 -> 압축을 푼 파일에서 옮기고 싶은 파일을 복사한다 -> 빈 폴더에서 붙여놓고 싶은 위치에 있는 폴더(파일)를 삭제하고 붙여넣는다. -> sourcetree에서 커밋 후 푸쉬하면 끝.
