데이터링크 계층

L2 SWITCH

LAN 은 고속통신을 목표로 한다
CSMA/CD 이 데이터 전송 방식 때문에 LAN이 고속으로 데이터를 전송가능하다.

CSMA는 한 쪽에 통신양이 많아서 끊기면 다른 쪽에 통신도 일정시간 동안 통신되지 않는다.
CD 찾아보기


1 OSI 모델중 2계층 모델이 하는 역할에 대해서 설명하세요


데이터 링크 2계층 : 데이터 링크 계층

역할 : 포인트 투 포인투 간 운송방식 결정.
장치간 데이터 운송방식 결정.

프로토콜 : LAN(이더넷), WAN(HDLC,PPP ...)

장치 : L2 SWITCH

2 데이터 링크의 목적달성을 위한 필요 요소 3가지는 무엇인가요

연결된 장비간 식별 주소
오류 제어
흐름 제어

3 LAN/WAN 의 대표 2계층 프로토콜의 명칭은 무엇인가요

이더넷,HDLC

4 초기 이더넷 프로토콜의 문제점은 무엇인가요
초기 이더넷 프로토콜 == CSMA/CD
데이터 통신량이 많을때 Collision Domain 이 발생 -> 일시적으로 모든 장치 통신 불가능

5 초기 이더넷 프로토콜이 사용하는 통신방법은 어떤것인가요

CSMA/CD

6 초기 이더넷 프로토콜의 문제점을 해결하기 위한 장치는

L2SWITCH

CSMA/CD 에서 L2SWITCH 로 발전.
