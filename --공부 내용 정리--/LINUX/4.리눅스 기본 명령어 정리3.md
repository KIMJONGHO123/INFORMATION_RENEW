# 리눅스 기본 명령어 정리3

## `cat` : 문서 전체 출력

### 옵션

- `n`  : 행 번호 출력

### 실습 예제

```bash
cat /etc/passwd
cat -n /etc/passw
```

---

## `head` & `tail` : 파일의 위쪽과 아래쪽에서 시작하여 출력 (기본 10줄)

### 옵션

- `n 숫자` : 지정된 숫자 줄수만큼 출력

### `head` 실습 예제

```bash
head /etc/passwd          # 기본 10줄 출력
head -5 /etc/passwd       # 상위 5줄 출력
head -2 /etc/passwd       # 상위 2줄 출력
```

### `tail` 실습 예제

```bash
tail /etc/passwd          # 기본 10줄 출력
tail -5 /etc/passwd       # 하위 5줄 출
```

---

## `more` : 화면 크기만큼 출력

### 실습 예제

```bash
more /etc/passwd
```
