ROUTING PROTOCOL
: 통신용 프로그램

Route : 경로,길
Router : 경로안내장치
Routing : 경로 안내작업, 경로안내중 ..

ROUTING PROTOCOL 은 정적과 동적으로 나뉨.

1) 동적은 전체를 파악하고 최적의 경로를 선택한다.

	네트워크 변화에 민감하게 대응할 수 있다.


2) 정적은 내가 직접 목적지와 방향을 입력

	네트워크 변화에 민감하게 대응하지 못한다.
	장치의 부하가 적고 속도가 빠르다

정적은 static 과 default로 나뉜다

static 단일 목적지에 대한 것만 입력

default는 내가 선택한 목적지 외에 모든 경로를 직접 저장.


정적 라우팅 내부 평가에 나온다.

가장 자리 라우터에만 디폴트 라우터 하면 된다.






IP넣은 거는 캡쳐 안해도 된다.

라우터의 static작업 올바르게 들어갔는지 캡쳐 1장나옴
결과에 따른 라우팅 테이블 1장나옴
한쪽에서 핑 테스트해서 끝에 컴퓨터와 잘 통신이 되는지. 4장 나옴
