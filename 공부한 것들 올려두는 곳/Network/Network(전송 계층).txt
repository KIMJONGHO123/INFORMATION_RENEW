전송 계층
장치 : L4 Switch(로드밸런서)

계층이 올라갈 수록 복잡해진다.

4계층은 밑에 하위 계층과 위의 상위 계층을 연결하는 역할을 주로 한다.

상위 계층은 운영체제에서 주로 담당한다.

END To END 데이터 전송 관련

송수신간 데이터가 잘 전달 됬는지 확인해주는 역할

TCP 프로토콜의 역할 중요중요중요

UDP : 비연결 서비스 예) 스트리밍

TCP : 연결지향 Protocol
UDP : 비연결지향 Protocol


1 OSI 모델중 4계층 모델이 하는 역할에 대해서 서술하세요 

종단간(송수신지사이) 데이터의 신뢰성을 보장한다.
 == 송수신지간에(END TO END)의 신뢰성 있는 데이터 전송을 담당

2 4 계층 관련 프로토콜은 어떤게 있고 각각 특징을 서술하세요


TCP : 연결지향형 프로토콜 / 데이터 신뢰도가 상대적으로 높다 / 속도가 느리다.


UDP : 비연결지향형 프로토콜 / 데이터신뢰도가 상대적으로 낮다. / 속도가 빠르다

최근에는 하드웨어의 발전으로 UDP써야하는 것도 TCP로 바꿔서 사용하는 경우가 많다.
