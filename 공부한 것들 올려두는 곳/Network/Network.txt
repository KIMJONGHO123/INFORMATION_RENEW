1. Network 는 무엇이며 무엇을 목적으로 하는가?

answer :  데이터교환을 위해 통신장치를 연결한 통신망.

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다

answer : LAN/WAN

3. LAN 의 특징에 대해서 서술하세요

answer : LAN은 근거리 통신망이며 비교적 근거리에 있는 장치들과 연결한다.

4. WAN의 특징에 대해서 서술하세요

answer : WAN은 원거리통신망(광역통신망) 이며 LAN과 달리 비교적 장거리에 있는 장치들과연결하는 것들 뜻한다.

5. InterNet 이란

answer : 한 특정집단끼리 연결하는 것이 아닌 전 세계적으로 연결된 컴퓨터 네트워크의 집합이다.
           여러 종류의 통신망을 한데 묶은 거대 통신망이다.

6. 프로토콜은 무엇인가?

answer : 프로토콜은 일정의 통신 규약이다. 즉 장치들간에 통신을 하기위해서 만든 규칙이다.


7. 네트워킹 모델의 목적은 무엇인가?

answer : 네트워킹 모델은 네트워크를 좀더 효율적으로 구축하고 설계하는데 필요한 설계도이다.
            통신망 구현을 위한 지침서. 하자 발생시 참고하는 지침서.

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요

answer : TCP/IP 모델은 물리 계층, 데이터 링크 계층, 인터넷 계층, 전송 계층 ,애플리케이션 계층이 있고 , OSI 7계층은 물리 계층, 데이터링크 계층, 네트워크 계층, 전송 계층 ,세션 계층, 프레젠테이션 계층, 애플리케이션 계층 등이 존재한다.

TCP/IP 각 계층별 역할

물리 계층 : 물리적인 연결과 전기 신호 변환/제어를 담당한다.

데이터링크 계층 : 네트워크 기기 간 데이터 전송 및 물리 주소를 결정한다.

네트워크 계층 : 다른 네트워크와 통신하기 위한 경로를 설정하고 논리 주소를 결정한다. IP 주소라는 개념이 이 계층에 존재한다.

전송 계층 : 네트워크 계층에서 보내온 데이터 정렬 , 오류 정정 등을 수행하고 신뢰할 수 있는 통신이다. 

애플리케이션 계층 : 통신하기 위한 필요한 서비스를 제공한다. 예(이메일, 웹 브라우저, 파일 전송 등)



OSI 7 계층 역할 : 

물리 계층 : 전송매체를 통한 데이터 전송 및 데이터의 신호 변환 예) 케이블 - 이더넷 케이블

데이터링크 계층 : MAC 주소를 사용하며 신뢰성 있는 정보를 전송하기 위한 역할을 수행한다.

네트워크 계층 : 데이터 전송 시 경로제어 및 인터네트워킹

전송 계층 : 송신 측과 수신 측의 연결 설정 및 유지

세션 계층 : 응용프로그램들 간의 통신관리 및 동기화 유지

프레젠테이션 계층 : 데이터의 번역 및 암호화

애플리케이션 계층 : 응용프로그램과 연계하여 사용자의 편리한 환경 제공
